# Android
### 项目名称
   省省
 
### 项目内容
 * 主打功能
      > 此APP用于购买者能在出门之前，就可以了解到购物商场的商品的具体优惠，方便购物无目标者快速找到商店里的优惠打折商品，同时方便有目标购物者用最优惠的价格买到目标商品。
      
   **实现工具**
 
 * 客户端   -
    * android studio   

 * 服务器端   
    * eclipse 
    * tomcat
 * 服务端框架    
    * spring+springMVC   
 * 数据库   
    * MySQL   
 * 数据传输框架    
    * Gson   
 * 页面原型   
    * 墨刀
